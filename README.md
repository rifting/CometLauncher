  <h1>Comet Launcher</h1>

  <p>An open-source, basic launcher for particular UE4 games</p>

---

Comet is a simple egui-based launcher meant to launch older verisons of particular UE4 games, for example, Fortnite.

It aims to be an alternative to [Reboot Launcher](https://github.com/Auties00/Reboot-Launcher).

[Nocturno](https://github.com/Windermed/Nocturno), [LawinServer](https://github.com/Lawin0129/LawinServer), and [Tellurium](https://github.com/plooshi/Tellurium) are some examples of projects that can be used with Comet Launcher.

## Using
Simply run the .exe and configure the game path and DLL paths!

## Compiling
`cargo build --release`

Find your binary in target/release/cometlauncher.exe !
